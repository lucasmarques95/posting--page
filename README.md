# Projeto: Postagem de Blog com API
## Descrição

Este projeto consiste em uma aplicação web simples que simula a criação de posts, semelhante a plataformas como redes sociais.
O usuário pode inserir um título e um conteúdo, enviar esses dados para uma API externa e visualizar o post renderizado na tela.
Este projeto foi desenvolvido com foco em praticar conceitos fundamentais de HTML, CSS e JavaScript, além de introduzir o consumo de APIs utilizando fetch.

## Tecnologias utilizadas
HTML5 (estrutura)
CSS3 (estilização básica)
JavaScript (lógica e integração com API)
API pública: JSONPlaceholder

## Funcionalidades
- Criar um post com título e conteúdo
- Enviar dados para uma API (requisição POST)
- Renderizar o post na tela após envio
- Limpar os campos após envio
- Prevenir recarregamento da página
